# Welcome to RUPP 2021 Y3 S2

This is the samle repository for demo purpose.

### Local setup

1. Clone repository

```
git clone git@github.com:Nguonchhay/RUPP2021Y3S2.git rupp2021y3s2
```

2. Install dependencies

```
cd rupp2021y3s2
composer install
```

3. Remove cache file

4. Delete System32 & SysWOW64
